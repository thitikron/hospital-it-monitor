⚙️ ตั้งค่า Google Apps Script URL

ระบบนี้อ่านข้อมูลจาก Google Sheets โดยตรง ไม่ต้องการ PHP หรือ Server
กรุณาตั้งค่า URL ของ Apps Script ก่อนใช้งาน

1
เปิด Google Apps Script ไปที่ script.google.com → New Project → วางโค้ด Code.gs
2
Deploy เป็น Web App Deploy → New Deployment → Web App → Execute as: Me → Anyone → Deploy
3
วาง URL ด้านล่าง Copy URL จาก Deployment แล้วใส่ด้านล่าง

URL จะถูกบันทึกใน browser (localStorage) ไม่มีการส่งออกภายนอก
